### Core Features

**1. Phonetic Table Display**
- Large conversion chart with all Pinyin/Zhuyin combinations
- Row headers (initials) and column headers (finals) always show Pinyin
- Color-coded groups for finals (a-group, o-group, i-group, u-group, Ã¼-group)
- Empty cells for invalid combinations

**2. Collapsible Settings Panel**
- Toggle between display modes: Zhuyin only, Pinyin only, or Both
- Highlight Gotchas toggle with category checkboxes (see below)
- Clean, minimal UI that collapses out of the way

**3. Text-to-Speech (Browser-Based)**
- Click any cell to hear the syllable pronounced in Mandarin Chinese
- Uses first tone (no tone marks displayed in zhuyin)
- Prioritizes Taiwan or Mainland Mandarin voices
- Graceful error handling if no suitable voice found
